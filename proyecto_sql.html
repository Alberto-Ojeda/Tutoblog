JAVA scene builder jpicker

Se realizara una aplicación en java usando el IDE eclipse y nuestro driver sql para poder entrelazar la aplcación en cuestión y nuestra base de datos. para la realización de nuestra interfaz utilizaremos scene builder y para la declaración de tablas usamos SQL y workbench como gestor.

Lo primero a analizar es nuestro SQL, en la imagen de abajo podemos ver que contiene 4 tablas o identidades estas tablas son, FACTURAS, PRODUCTOS, CLIENTES, FACTURAS_PRODUCTOS, ¿ como estan conectadas ?

La relación que tienen estas identidades son las siguiente:

  • PRODUCTOS a FACTURAS_PRODUCTOS: la relacón entre estos dos es de 1 a muchos, es decir para 1 productos puede existr una o mas facturas
  • FACTURAS_PRODUCTOS a FACTURAS: estas dos identidades estan relaciionadas debido a que las facturas contienen la información de los clientes y por otra parte FACTURAS_PRODUCTOS pasara los datos y cantidades de los productos adquridos, su relación es de muchos a 1 es decir una factura puede contener factura de cliente puede tener varias facturas de productos
  • CLIENTES a FACTURAS: por ultimo su relación entre estas identidades son 1 cliente puede tener varias facturas
